#  ABC Bank – Django Web Application

##  Project Overview
ABC Bank is a full-featured banking web application built using Django. It allows users to securely manage accounts, perform transactions, and view financial activities through a user-friendly interface.

---

##  Features

###  User Features
- User Registration & Login
- Secure Authentication System
- Profile Management (Edit Profile, Upload Profile Picture)
- View Account Details & Balance

###  Banking Operations
- Deposit Money
- Withdraw Money
- Fund Transfer Between Accounts(email)
- Transaction History with Credit/Debit Indicators



##  Security Features
- CSRF Protection
- Form Validation
- Secure Authentication System



##  Tech Stack

- **Backend:** Django (Python)
- **Frontend:** HTML, CSS, Bootstrap
- **Database:** SQLite (Development)
- **Version Control:** Git & GitHub
